# Included Projects

## 1. Gene2protein

### Description

A bioinformatics project for translating **DNA sequences into protein sequences** using BioPython.

### Concepts Covered

- DNA sequence processing
- Codon translation
- Amino acid generation
- Biological sequence validation

### Main Files

```text
translator.py
sample_sequence.txt
protein_result.txt
```

---

## 2. PCA_Gene_Expression

### Description

A project that performs Principal Component Analysis (PCA) on multidimensional biological expression datasets.

### Concepts Covered

- Dimensionality reduction
- Gene expression analysis
- Data visualization
- PCA transformation

### Main Files

```text
dimensionality_reduction.py
cellular_expression.csv
expression_projection.png
```

---

## 3. GO_Enrichment

### Description

A simulated Gene Ontology enrichment analysis project for identifying biological functional categories associated with selected genes.

### Concepts Covered

- Functional enrichment
- Biological interpretation
- Pathway analysis
- Data visualization

### Main Files

```text
biological_interpretation.py
selected_genes.txt
functional_summary.csv
```

---

## 4. ViterbiHMM

### Description

A computational biology project implementing Hidden Markov Models and the Viterbi algorithm for hidden state prediction.

### Concepts Covered

- Hidden Markov Models
- Dynamic programming
- Sequence prediction
- Probability analysis

### Main Files

```text
state_prediction.py
symptom_sequence.txt
final_prediction.txt
```
